# Phase 2 — SQL & PySpark
Learned:

- DataFrame Creation → createDataFrame(), spark.read.csv()
- Viewing Data → show()
- View structure of DataFrame → printSchema()
- Selection & Filtering → select(), filter() / where()
- Aggregation → groupBy(), sum(), count(), avg()
- Sorting → orderBy()
- Create/modify column → withColumn()
- Remove column → drop()
- Get unique values → distinct()
- Rename column → withColumnRenamed()
- Column reference → col()
- Conditional update → when()
- Rounding values → round()
- Join DataFrames → join()
- SQL Integration → createOrReplaceTempView(), spark.sql()

Challenges/Further Improvements:

- Understanding when to use → join() vs direct aggregation
- Handling data types correctly → cast()
- Avoiding precision issues → float vs double
- Debugging unexpected outputs in joins and aggregations
